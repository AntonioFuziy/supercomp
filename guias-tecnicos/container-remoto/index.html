    Tutorial Acesso à maquina Monstrão via VsCode¶

    • Você deve ter acesso a sua maquina virtual no monstrão e o VsCode instalado.

    • Este Tutorial irá seguir as seguintes etapas.

    • Etapas:

      • Instalar extensão no VsCode;
      • Fazer acesso ssh;
      • Instalar o Jupyter notebook;
      • Rodar o notebook na pasta alvo.

    Instalar extensão SSH Remote no VScode¶

    • Dentro do aplicativo do VsCode Ctrl+Shift+X
    • Buscar e instalar os pacotes do Remote-SSH

    Acesso SSH¶

    • Clicar no icone a esquerda Remote Explorer
    • Adicionar um novo SSH TARGET
    • Na linha de comando digitar ssh user@<seu IP do monstrão>
    • Clicar Enter para ignorar o acesso via par de chaves
    • O VsCode irá conectar na sua Máquina virtual e pedir a senha
    • Digite a senha de acesso do seu usuário

    Como instalar e iniciar o Jupyter Notebook com Python no Ubuntu 20.04¶

    • Instalações iniciais:

      • sudo apt update
      • sudo apt upgrade -y
      • sudo apt install build-essential
      • sudo snap install jupyter

    • clonar via git, o repositório da disciplina

    • Acessar a pasta com o projeto e rodar o comando: jupyter notebook
    • Neste momento o Jupyter irá iniciar o servidor na máquina virtual e o VsCode irá mostrar uma mensagem perguntando se deseja abrir via Browser. Aceitar
    • Copiar o token gerado no terminal e colar na aba aberta no Browser.

    PRONTO.¶

    Outra dica¶

    • Se quiser fazer via Tunel SSH, pode rodar o comando abaixo no seu terminal e acessar o Browser na porta 8888

    ssh -L 8888:localhost:8888 user@< digite aqui ip da sua maquina virtual no monstrão > * Acesse o localhost:8888 no seu Browser.
